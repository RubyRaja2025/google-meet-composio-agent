# Google Meet Agent

AI agent for querying Google Meet meetings and Gemini notes via Composio OAuth + Anthropic Claude.

## Architecture

```
┌─────────────────────────────────────────────────┐
│          Google Meet Agent                       │
│  ┌───────────────────────────────────────────┐  │
│  │  1. Check for existing connection         │  │
│  │  2. If none → initiate OAuth flow         │  │
│  │  3. Get Google Meet tools from Composio   │  │
│  │  4. Execute queries via Claude            │  │
│  └───────────────────────────────────────────┘  │
└──────────────────────┬──────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────┐
│              Composio Platform                   │
│  (handles OAuth tokens, API calls, refresh)      │
└──────────────────────┬──────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────┐
│            Google Meet REST API v2               │
│  (conferenceRecords, participants, transcripts)  │
└─────────────────────────────────────────────────┘
```

## Usage

```python
from google_meet_agent import GoogleMeetAgent

agent = GoogleMeetAgent()
agent.setup()  # handles OAuth if needed

# Natural language query
response = agent.query("Show me my recent meetings")
print(response.data)

# Convenience methods
response = agent.list_conferences(limit=10)
response = agent.get_participants("conferenceRecords/abc123")
response = agent.get_transcript("conferenceRecords/abc123")
```

## Exception Hierarchy

```
GoogleMeetAgentError (base)
├── ConfigurationError      # Missing config
├── AuthConfigNotFoundError # Invalid auth config
├── OAuthTimeoutError       # User didn't complete OAuth
├── ConnectionExpiredError  # Needs re-auth
├── ComposioConnectionError # Composio connection failed
├── GoogleMeetAPIError      # API errors
│   └── RateLimitError      # 429 response
├── NoConferencesError      # No meetings found
└── AgentExecutionError     # Agent execution failed
```

## Google Meet Tools (READ-ONLY)

- `GOOGLEMEET_LIST_CONFERENCE_RECORDS` - List past meetings
- `GOOGLEMEET_GET_CONFERENCE_RECORD` - Get meeting details
- `GOOGLEMEET_LIST_PARTICIPANT_SESSIONS` - List attendees
- `GOOGLEMEET_GET_PARTICIPANT_SESSION` - Get participant details
- `GOOGLEMEET_GET_TRANSCRIPTS_BY_CONFERENCE_RECORD_ID` - Get transcript

## Google Drive Tools (for Gemini Notes)

- `GOOGLEDRIVE_LIST_FILES` - Search for meeting notes documents
- `GOOGLEDRIVE_DOWNLOAD_FILE` - Read file content
- `GOOGLEDRIVE_GET_FILE_METADATA` - Get file details

## Important Notes

- **READ-ONLY agent** - cannot create or modify meetings
- Requires **Google Workspace** account (Business Standard+)
- Personal Gmail has very limited API access
- Transcripts only available if enabled during meeting
- Transcript entries expire after 30 days
